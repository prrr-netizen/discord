실시간으로 바로 들어가는 HTML5 미니게임 룸

디스코드 계정 하나로 접속해서,
브라우저 안에서 바로 돌아가는 외부 HTML5 미니게임들을 한 곳에서 즐길 수 있는 허브입니다.

1

브라우저만 있으면 바로 실행

별도 설치 없이, PC·모바일 브라우저에서 바로 HTML5 미니게임을 플레이합니다.

2

외부 게임 임베드

Playpager 등 외부에서 제공하는 무료 임베드 게임을 그대로 끌어와 실시간 룸처럼 구성합니다.

3

디스코드 연동 포인트 구조

디스코드 봇과 연동하면, 각 게임 결과를 포인트·이벤트 구조에 연결해서 운영할 수 있습니다.

LIVE
GAME #1
온라인 체스 (예시)
Playpager에서 제공하는 체스 게임 임베드입니다.
LIVE
GAME #2
리버시 / 오델로 (예시)
다른 HTML5 퍼즐/보드 게임으로 쉽게 교체할 수 있습니다.
위 미니게임들은 예시이며, Playpager 또는 유사 서비스에서 제공하는 다른 HTML5 게임 iframe으로 언제든 교체할 수 있습니다.
